Bash - Select n arg from the last command
=========================================
//////
:copyright: 2013, Julien Fabre
:revinfo:
v1.0, Julien Fabre, 2013-11-06: Initial version
:convert: html
//////


[source,bash]
----
# Example
cp -p afile /path/to/adir/

# ls to /path/to/adir/
ls !$
# or
<ALT+.>
# or
# cat afile
cat !:2
----
