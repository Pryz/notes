Plugins
=======

Autoloaders
-----------

``` bash
tree autoload/
# autoload/
# ├── pathogen.vim
# └── snipMate.vim
```

Bundles
-------

``` bash
tree -L 1 bundle/
# bundle/
# ├── nerdtree
# ├── vim-endwise
# ├── vim-puppet
# └── vim-ruby
```

Settings
========

``` bash
# <CTRL+w> + <Arrows> -> change the current buffer/window
:set expandtab
:set tabstop=4
```

Keys
====

``` bash
vib # select block
viB # select curly brake
vi" # select text in " "
{ d} # cut paragraph
} # go to the next blank line
% # jump to the matching { }, ( )
```
